Chapter - 2 Topics

Bold

Bold (<b>): Displays text in bold without extra emphasis.

Strong

Strong (<strong>): Displays text in bold with semantic importance.

Italic

Italic (<i>): Displays text in an italic style.

Emphasis

Emphasis (<em>): Displays text in italics with semantic emphasis.

abbreviation

HTML Hyper Text Markup Language (<abbr>): Indicates an abbreviation, often with a tooltip on hover.

Citation

Citation (<cite>): Displays the title of a work in italics, representing a citation.

Defination

Definition (<dfn>): Represents a definition term, typically italicized.

Insert

Insert (<ins>): Shows inserted text with an underline.

Delete

Delete (<del>): Displays deleted text with a strikethrough.

Strikethrough

Strikethrough (<s>): Renders text with a line through it, indicating incorrect or irrelevant text.

Quotation

Quotation (<q>): Displays inline quotations, often enclosed in quotation marks.

Superscript

Superscript (<sup>): Positions text slightly above the normal line.

Subscript

Subscript (<sub>): Positions text slightly below the normal line.
